# Healthcare of Technology - Website Portofolio

Website portofolio dengan tema kesehatan yang fokus pada edukasi pengurangan konsumsi gula.

## Fitur Utama
- Website responsif yang bekerja di semua perangkat
- Navigasi smooth antara section
- Menampilkan alternatif minuman dan makanan rendah gula
- Chart nutrisi interaktif menggunakan Chart.js
- Animasi counter untuk menunjukkan pengurangan gula
- Desain dengan warna sesuai permintaan (#1B3C53, #234C6A, #456882, #E3E3E3)
